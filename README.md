## üìù Example Usage

### Get All Books
To retrieve a list of all books, send a GET request to:
```
GET http://localhost:8080/api/books
```

### Add a New Book
To add a new book, send a POST request with the following JSON body:
```json
{
  "title": "New Book Title",
  "author": "Author Name",
  "publishedDate": "2023-01-01"
}
```
